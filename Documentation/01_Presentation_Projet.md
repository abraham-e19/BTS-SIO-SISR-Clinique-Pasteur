# 1. Présentation du projet

## 1.1 Contexte général
Cette réalisation professionnelle s’inscrit dans le cadre du projet « Clinique Pasteur » du BTS Services Informatiques aux Organisations, option SISR.
L’objectif du projet est de mettre en place une infrastructure informatique sécurisée répondant aux besoins d’une clinique, tout en respectant les exigences de confidentialité et de protection des données (RGPD).

Dans ce contexte, la direction des systèmes d’information (DSI) a demandé la mise en œuvre d’un serveur de base de données sécurisé destiné à héberger les données applicatives de la clinique.

## 1.2 Objectifs techniques
Les objectifs techniques de cette réalisation sont les suivants :
- Installer un serveur Debian en mode texte afin de garantir stabilité et sécurité
- Déployer un serveur de base de données pour centraliser les données
- Sécuriser les accès au serveur et aux données
- Mettre en place un système de sauvegarde automatique
- Fournir une documentation technique complète et exploitable

## 1.3 Périmètre de la réalisation
La réalisation couvre :
- la création d’une machine virtuelle sur Proxmox
- l’installation du système Debian sans interface graphique
- l’installation et la configuration du serveur de base de données
- la sécurisation du serveur (pare-feu, comptes, droits)
- la mise en place de sauvegardes
- les tests de bon fonctionnement

La partie développement applicatif n’est pas incluse dans cette réalisation.
